# Trader Behavior vs Market Sentiment Analysis

## Objective
Analyze how trader behavior and performance vary across different
market sentiment conditions (Fear vs Greed).

## Datasets
- Historical Trader Data (Hyperliquid)
- Bitcoin Fear & Greed Index

## Tools
- Python
- Pandas
- Matplotlib
- Google Colab

## Project Structure
- notebook_1.ipynb: Complete analysis
- csv_files/: Raw and processed datasets
- outputs/: Saved charts and visualizations
- ds_report.pdf: Final insights and conclusions

## Key Insights
- Profitability patterns differ under Fear and Greed
- Trade sizes tend to increase during Greed phases
- Loss frequency is higher during Fear-dominated markets
